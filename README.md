# E-Commerce-Business-Insights-Dashboard
This project presents an interactive Power BI dashboard analyzing e-commerce sales performance

## Key Metrics
- Total Sales
- Total Profit
- Total Orders
- Average Order Value

## Dashboard Features
- Monthly Sales Trend Analysis
- Top Performing Products 
- Sales Distribution by Category
- Regional Sales Insights
- Customer Segment Analysis

## Tools Used
- Power BI
- DAX
- Data Visualization
